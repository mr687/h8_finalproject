# Hactiv8 - Final Project (Admin Panel E-Commerce)

## Preparation
- Clone this repo
- `composer install`
- `cp .env.example .env` & Set-up database
- `php artisan key:generate`
- `npm install && npm run dev`
- `php artisan serve`
